## 如何记录

每次有设计决策、架构讨论或重要的实现对话时：

1. 在 `logs/conversations/` 下创建 `YYYY-MM-DD_HHMMSS_topic.md`
2. 记录：讨论内容、关键决策、代码产出、下一步计划
3. 在本文件中添加索引

## 记录模板

```markdown
# YYYY-MM-DD HH:MM — 主题

## 讨论内容
...

## 关键决策
- ...

## 代码产出
- `path/to/file` — 说明
```

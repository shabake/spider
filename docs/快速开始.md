# 快速开始

## 安装

```bash
# 克隆项目
git clone <repo-url>
cd spider

# 安装核心依赖
pip install openai pyyaml rich

# Web UI 需要额外安装
pip install fastapi uvicorn

# 文档转换功能（可选）
pip install python-docx PyMuPDF fpdf2
```

## 设置 API Key

```bash
# 推荐：环境变量
export DEEPSEEK_API_KEY="sk-xxxx"

# 或者：写入 shell 配置（一劳永逸）
echo 'export DEEPSEEK_API_KEY="sk-xxxx"' >> ~/.zshrc
source ~/.zshrc
```

!!! tip "环境变量参考"
    | 变量 | 默认值 | 说明 |
    |------|--------|------|
    | `DEEPSEEK_API_KEY` | — | DeepSeek API Key（必填） |
    | `DEEPSEEK_BASE_URL` | `https://api.deepseek.com/v1` | API 地址 |
    | `DEEPSEEK_EMBEDDING_MODEL` | `deepseek-embedding` | Embedding 模型 |

## 启动

### Web UI（推荐）

```bash
python main.py --web
# 打开 http://127.0.0.1:8888
```

### 交互式 CLI

```bash
python main.py -i
# 进入后看到 > 提示符，直接输入任务
```

### 单次任务

```bash
python main.py "当前目录有哪些文件？"
```

## 第一个任务

启动后，试试这些任务：

```bash
# 系统信息
python main.py "我的电脑还剩多少磁盘空间？"

# 文件操作
python main.py "在桌面创建一个 hello.txt，写入 Hello Spider"

# 并行查询
python main.py "分别查一下系统版本、磁盘空间、内存使用"

# 保存技能
python main.py "查一下系统运行时间，然后把方法保存为技能"
```
